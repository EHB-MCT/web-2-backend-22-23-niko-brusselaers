• GET:

    • /

      return page with documentation of endpoints

    • /getRandomGame

      return a random game

    • /getGameByPreferences

      return a random game based on a couple of questions

    • /getGameListFromFavorites

      return a random game from the list of favorited games

    • /getUserPreferences

      return list of likes and disliked games of user

  • POST:

    • /createAccount

      create a new user account and add it to the database

    • /login

      check user login credentials and let them login

    • /loginId

      function to login with userId

    • /addUserGamePreference

      add game preference to user game preferences list in the database

  • PUT:

    • /updateAccount

      update user credentials in database

  • DELETE:

    • /deleteAccount

      delete user account from database

    • /deleteUserGamePreference

      remove game preference from user game preferences list in the database
